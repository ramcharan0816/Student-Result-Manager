# Student Result Web App Manager

A web-based application designed to manage student academic results efficiently. This app allows users to add, update, and view student marks through an interactive and user-friendly interface.

--> Features

* Add student details and subject marks
* Update and delete student records
* Automatic result calculation
* Display results dynamically
* Simple and responsive UI

--> Technologies Used

* HTML
* CSS
* JavaScript

--> Usage

Open the application in a browser, enter student information, and manage results easily without any backend.
